# pwa_rangsit
คู่มือและเอกสารประกอบตามโครงการติดตั้งตู้สาขา (PABX) ขนาด 4 สายนอก 16 สายใน
- CS user manual.pdf = เอกสารคู่มือฉบับเต็ม (ภาษาอังกฤษ)
- Thai_Manual PABX CS416-432.pdf = เอกสารคู่มือฉบับย่อ ภาษาไทย
- index.html เว็บนำเสนอการตั้งค่าตู้สาขา PABX สำหรับ กปภ. สาขารังสิต
- PWA_Rangsit_Config.txt = ขั้นตอนการตั้งค่า PABX Config สำหรับ การประปาภูมิภาค สาขารังสิต
- overview.html เว็บแสดงภาพองค์ประกอบหลักของตู้
- user.html เว็บนำเสนอวิธีการใช้งานของผู้ใช้งานทั่วไป
- pages2.png, pages3.png, pages4.png เป็นรูปภาพจากเอกสารคู่มือที่แสดงภาพองค์ประกอบหลักของตู้
