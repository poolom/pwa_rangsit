📘 คู่มือการใช้งานระบบโทรศัพท์ CS416 / CS+

📖 วิธีใช้งานพื้นฐานสำหรับผู้ใช้ทั่วไป (User Manual)
📞

การโทรออก

  • #1M โทรออกตามช่องสายนอก
    M = 1-6 (เลือกช่องสายที่ต้องการ)
  • ยกหู + เบอร์ภายใน โทรภายใน
    กดหมายเลขเครื่องที่ต้องการ (หากเปลี่ยนเบอร์ให้กดเบอร์ใหม่)
  • ตั้งค่าห้ามโทรออก ถ้าตั้งห้ามโทรออก จะไม่สามารถใช้สายนอกได้
📲 [ รูปภาพ: การกดหมายเลขโทรออก ] ➡️ ☎️
🔄

การโอนสาย

  • Flash + เบอร์ โอนสายทันที
    (ถ้าไม่มีปุ่ม Flash ให้วางหูแล้วกดหมายเลข)
  • #81 + เบอร์ โอนทันทีไปเบอร์ที่ต้องการ
    (ยกเลิกโดยกด #81#)
  • #82 + เบอร์ โอนเมื่อไม่ว่าง / ไม่มีคนรับ
    ระบบจะโอนไปที่เบอร์ที่ตั้งไว้ (ยกเลิก #82#)
🔀 [ รูปภาพ: การกดปุ่ม Flash ] ➡️ 📤
⚙️

ฟังก์ชันเสริม

  • #9 ตั้งสายมาคุยแทน (เมื่อสายภายในไม่มีใครรับ)
  • #9 + Ext ตั้งสายมาคุยจากเบอร์ที่ต้องการ (Ext = เบอร์ภายใน)
  • Flash + * + เบอร์ ประชุมสาย (ไม่ต้องวางหู)
    ใช้กับสายนอก 1 และ สายใน 2 คู่สายคุยกัน
  • #83 ไม่ต้องการรับสาย (ยกเลิกโดยการยกหู)
🎙️ [ รูปภาพ: การประชุมสาย / ตั้งค่า ] ➡️ 👥
📋

ตารางสรุปคำสั่งลัดสำหรับผู้ใช้

คำสั่ง / การกระทำ ฟังก์ชัน หมายเหตุ / การยกเลิก
#1M (M=1-6) โทรออกตามช่องสายนอกที่ต้องการ เลือกช่องสายเฉพาะ (1-6)
ยกหู + เบอร์ โทรภายใน กดเบอร์ภายในที่เปลี่ยนใหม่ได้
Flash + เบอร์ โอนสายไปยังหมายเลขอื่น ถ้าไม่มีปุ่ม Flash ให้วางหูแล้วกดเบอร์
#81 + เบอร์ โอนสายทันทีไปเบอร์ที่ต้องการ ยกเลิกโดยกด #81#
#82 + เบอร์ โอนสายหากไม่มีผู้รับ / สายไม่ว่าง ยกเลิกโดยกด #82#
#9 หรือ #9 + Ext ตั้งสายมาคุยแทน ใช้เมื่อสายภายในไม่มีใครรับ
Flash + * + เบอร์ ประชุมสาย ไม่ต้องวางหู ใช้กับสายนอก 1 และสายใน 2
#83 ไม่ต้องการรับสาย (DND) ยกเลิกโดยการยกหูโทรศัพท์
🖼️ [ รูปภาพ: ตารางคำสั่งหรือแผนภาพการใช้งาน ] User Guide
💡 เคล็ดลับ: หากโทรศัพท์ไม่มีปุ่ม Flash ให้ใช้วิธีวางหูแล้วกดหมายเลขใหม่เพื่อโอนสายแทน
